# AI Dashboard Generator

An AI-powered web application that converts raw datasets into interactive dashboards and generates automated business insights.

## Features
- Upload CSV/Excel files
- Automatic data visualization
- Interactive charts (Plotly)
- AI-generated insights
- Company logo integration

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- OpenAI API

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
